# HR Sim Remastered — Office Chaos

A browser-based HR management tycoon. Grow your company, outpace rivals, and hit your market share target before you go bankrupt.

## Play

```bash
cd web
npm install
npm run dev
```

Open http://localhost:5173

## Legacy

The original Python/tkinter versions are kept for reference:

- `new_sim.py` — v2 (recommended legacy version)
- `manager_sim.py` — v1 prototype

Run with `python3 new_sim.py`

## Stack

Vite · React · TypeScript · Tailwind CSS · Framer Motion
