# Office Chaos — HR Tycoon

A polished browser-based HR management simulator. Build a company, beat the market, and don't go broke!

## Quick Start

```bash
cd web
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## How to Play

- **Choose** your company type, leadership style, and difficulty
- **Each tick** = 1 month — you earn profit (revenue − salaries)
- **Spend** on actions (training, bonuses, marketing) to boost stats
- **Buy** upgrades and departments for permanent passive bonuses
- **Win** by reaching the market share target before going bankrupt
- **Lose** if cash, employees, or reputation hit zero

## Tech Stack

- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion for animations
- localStorage for save/load
